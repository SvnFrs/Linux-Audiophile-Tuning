starting and stopping of the MPD server. You will therfore need to ensure
that MPD is already configured and running.

-. Top-level music path: > 상위 음악 경로:
-. The 'Music folder' setting is used to lookup cover-art. It may be set to a HTTP URL if your MPD is on another machinen, and covers are accessible via HTTP. If it is not set to a HTTP URL, and you also have write permissions to this folder (and it's sub-folder), then Cantata will save any downloaded covers into the respective album folder. This folder will also be used to locate music files to transferring to (and from) devices.
   '음악 폴더' 설정은 음반표지를 찾기 위해 사용됩니다. MPD가 다른 기기에 있고 표지가 HTTP를 통해야 한다면, HTTP URL로 설정할 수 있습니다. HTTP URL로 설정되지 않고 이 폴더에 쓰기 권한이 있다면, 내려받은 표지를 각각의 음반 폴더에 저장합니다. 이 폴더는 장치로(부터) 전송하는 음악파일을 찾는데에도 사용됩니다.
-. If no setting is specified for 'Cover filename', then Cantata will use a default of cover
   '표지 파일이름'이 설정되지 않으면, 기본 표지를 사용합니다
-. If your MPD is configured to use a UPnP/DLNA backend, then 'Top-level music path' should be set to the 'All Music' path (e.g. servername\Music\All Music\) to prevent duplicate tracks. If this setting is used, then Cantata will only list tracks that are under this MPD path. For most MPD setups, this setting should be left blank. Changes to this setting will require a full refresh of the database.
   MPD가 UPnP/DLNA를 사용하도록 설정되었다면, 곡들이 겹치지 않도록 '상위 음악 경로'는 '전체 음악' 경로 (예. 서버이름\음악\전체 음악)로 설정되어야 합니다. 설정하면 이 MPD 경로의 곡들만 나옵니다. 대부분의 경우, 이 설정은 비워두어야 합니다. 설정을 바꾸면 데이터베이스의 전체 새로 읽기를 해야 합니다.
-. Standard - music collection may be shared, is on another machine, is already setup, or you wish to enable access from other clients (e.g.MPDroid) Basic - music collection is not shared with others, and Cantata will configure and control the MPD instance. This setup will be exclusive to Cantata, and will <b>not</b> be accessible to other MPD clients.") NOTE: If you wish to have an advanced MPD setup (e.g. multiple audio outputs, full DSD support, etc) then you <b>must</b> choose 'Standard'
   표준 - 다른 기기에서 이미 설정된 음원을 공유하거나, 다른 클라이언트(예. MPDroid)에서 접속할 수 있습니다. 기본 - 음원을 공유하지 않고 칸타타가 MPD를 설정하고 제어합니다. 이것은 칸타타만의 설정으로 다른 MPD 클라이언트에서는 접속이 <b>안</b>됩니다. 참고: MPD의 고급 설정을 하려면(예. 다중 오디오 출력, DSD 지원 등) <b>반드시</b> '표준'을 선택해야 합니다.
