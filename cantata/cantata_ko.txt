버전 2.3

* 바뀐 점

 1. 내장 HTTP 서버로 로컬 파일 재생 (마우스 끌어 놓기/오른쪽 키, 홈/루트 폴더)
 2. Avahi로 MPD 서버 검색.
 3. 서버 음원 자동 업데이트.
 4. 음반 분류 보기.
 5. 툴바 마우스 오버 음반표지 업데이트.
 6. 격자 음반표지 크기 조절 (Ctrl +/-)
 7. 정보의 음반표지 늘이기 옵션 추가.
 8. 곡 정보 보기 (파일 형식, 샘플/비트레이트)
 9. 정보의 가사 버그 수정.
10. 윈도판 글꼴 개선과 최대화 상태 저장.
11. 창의 가로 크기에 따른 UI 자동 변경.
12. forked-DAAP 지원.

* 번역

1. 에피소드를 읽은 > 에피소드를 들은
2. %artist%은 > %artist%는
3. Play Queue > 연주순서
4. Playlist > 연주목록
5. 로컬 파일 재생 : 내장 HTTP 서버, 자동 업데이트 : 서버 자동 감지

MPD는 대게 음원의 곡들만 재생합니다. 최근 MPD는 (v0.19 이후부터) 로컬 MPD 서버에 연결되어 있으면 다른 파일도 재생할 수 있습니다(예. 주소가 '127.0.0.1' 또는 'localhost'). 이를 위해서는 MPD 서버에서 해당 파일의 읽기 권한이 있어야 합니다. 이전 MPD에서 로컬 MPD가 아니거나 다른 사용자가 로컬 파일에 접근할 수 없으면, 칸타타의 간단한 HTTP 서버를 이용해도 됩니다. 이 서버는 실행되는 칸타타의 연주순서에서만 파일을 불러올 수 있습니다.
